# Google Form Svelte Clone

## Installation

```
npm install
cd server
npm install
```

## Start

```
npm run dev -- --open --host
# on an other shell
cd server
node src/index.js
```

> Application will be running on localhost:3000

> Server on localhost:5000

## Docker Compose
```
npm run compose
```
> Application prod builded running on port 80 

> Server on port 5000
